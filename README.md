## 1. Introduction
GHURBO is a **full-stack travel platform** built for Bangladesh. Travelers can discover curated tour packages, compare options, chat with an AI assistant, and book securely ‚Äî while tour providers manage listings, pricing, and bookings through a dedicated dashboard.

## 2. Features
- **üéØ Curated Packages** ‚Äî Filter by destination, budget, and dates with live availability.
- **üíé Exclusive Offers** ‚Äî Seasonal discounts and special deals.
- **ü§ñ AI Assistant** ‚Äî English & Bangla chatbot for instant help and recommendations.
- **üìä Provider Dashboard** ‚Äî Manage packages, bookings, and visibility.
- **üîí Secure Booking & Emails** ‚Äî Reliable confirmations and receipts.
- **üì± Responsive UI** ‚Äî Mobile-first, modern interface.
